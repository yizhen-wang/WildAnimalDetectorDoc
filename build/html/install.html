Installation¶

WildAnimalDetector supports multiple operating systems including Linux, Windows and MacOS.

Detectron2¶

Detectron2 is Facebook AI Reseach’s next generation library that provide state-of-art detection and segmentation algorithms. You can refer to their installation instructions. See Detectron2

Other Packages¶

You also need to install the following packages, you can install these packages with pip3:

pip3 install opencv
pip3 install Pillow
pip3 install imantics
pip3 install scikit-imag
pip3 install ensemble-boxes
